Superhero Shodown

Superhero Shodown is a card-battling app made using Angular 7 and PostgreSQL. Players choose heroes from the Marvel and DC universes to see if they can defeat their opponents. The heroes are separated by attack type which players use to gain a strategic advantage in battle. This project I built with my team won Best Front-End project at Grand Circus' Demo Day on June, 28, 2019.

Blockbuster (... because it's come to this)

Blockbuster (... because it's come to this) is an movie searching app using Angular 7 and theMovieDB API. This was a fun app that I made with my team that imagined where Blockbuster would be now with the rise of streaming services. Movies can be searched by title, genre, year, or rating. Once searched, you can click on the Blockbuster icon to add the movies to the favorites page.
